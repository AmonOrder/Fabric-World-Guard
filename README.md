# FabricWorldGuard

[![License: GPL-3.0](https://img.shields.io/badge/license-GPL--3.0-blue.svg)](https://choosealicense.com/licenses/gpl-3.0/)
![Minecraft](https://img.shields.io/badge/Minecraft-1.20.1-green)

FabricWorldGuard — это мод для серверов на платформе Fabric, предназначенный для защиты территорий.
Он позволяет игрокам и администраторам создавать безопасные зоны, чтобы предотвратить нежелательное разрушение
построек и кражу ресурсов.

## Для кого этот мод?
* **Для владельцев серверов:** Чтобы защитить спавн и постройки игроков от гриферов.
* **Для обычных игроков:** Чтобы обезопасить свой дом и сундуки от посягательств других игроков.

## Возможности
* **Создание регионов:** Выделение территорий с помощью специальных инструментов (например, деревянного топора).
* **Управление флагами:** Запрет или разрешение PvP, взрывов, спавна мобов или открытия контейнеров.
* **Система владения:** Передача прав управления регионом другим игрокам.

## Список команд
Вот основные команды, доступные игрокам и администраторам:

| Команда                                        | Описание                                    |
|:-----------------------------------------------|:--------------------------------------------|
| `/region claim <имя региона>`                  | Создать регион на основе выделенной области |
| `/region flag <имя региона> <флаг> <значение>` | Установить флаг (например, `pvp false`)     |
| `/region addmember <имя региона> <игрок>`      | Добавить игрока в регион участником         |
| `/region removemember <имя региона> <игрок>`   | Удалить игрока участника из региона         |
| `/region addmowner <имя региона> <игрок>`      | Добавить игрока в регион совладельцем       |
| `/region removeowner <имя региона> <игрок>`    | Удалить игрока совладельца из региона       |
| `/region delete <имя региона>`                 | Удалить существующий регион                 |

## Требования
* **Minecraft:** 1.20.1 
* **Yarn Mappings** 1.20.1+build.10
* **Fabric Loader:** 0.15.11
* **Fabric API:** 0.92.0+1.20.1
* **World Editor** 7.2.15

## Установка
1. Скачай последнюю версию мода с GitHub Releases.
2. Убедись, что у тебя установлены Fabric Loader и Fabric API.
3. Помести файл `.jar` в папку `mods` на сервере или в клиенте.
