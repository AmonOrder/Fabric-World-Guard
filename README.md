# FabricWorldGuard

[![License: GPL-3.0](https://img.shields.io/badge/license-GPL--3.0-blue.svg)](https://choosealicense.com/licenses/gpl-3.0/)
![Minecraft](https://img.shields.io/badge/Minecraft-1.20.1-green)

FabricWorldGuard — это мод для серверов на платформе Fabric, предназначенный для защиты территорий.
Он позволяет игрокам и администраторам создавать безопасные зоны, чтобы предотвратить нежелательное разрушение
построек и кражу ресурсов.

## Для кого этот мод?
* **Для владельцев серверов:** Чтобы защитить спавн и постройки игроков от гриферов.
* **Для обычных игроков:** Чтобы обезопасить свой дом, сундуки и питомцев от посягательств других игроков.

## Возможности
* **Создание регионов:** Выделение территорий с помощью специальных инструментов (например, деревянного топора).
* **Управление флагами:** Запрет или разрешение на действия в регионе.
* **Система прав:** Передача прав управления регионом другим игрокам.

## Список флагов
Вот основные флаги, доступные игрокам и администраторам:
| Флаг              | Описание                                                                                                                                               |
|:------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------|
|enderman-grief     | Защита региона от эндерменов (установка и кража ими блоков), по умолчанию включено (false)                                                             |
|entry              | Защита региона от посещения других игроков (создаёт у региона барьер, не пропускающий игроков), по умолчанию отключено (true)                          |
|explosions         | Защита региона от взрывов (разрушение блоков через динамит и т.д.), по умолчанию включено (false)                                                      |
|hanging-protection | Защита рамок и картин от взаимодействия игроков без прав в вашем регионе, по умолчанию включено (false)                                                |
|mob-protection     | Защита ваших питомцев и иных мирных существ от урона сто стороны игроков без прав в вашем регионе, по умолчанию включено (false)                       |
|pvp                | Защита от урона между игроками на территории региона, по умолчанию выключено (true)                                                                    |
|use-chests         | Защита региона от использования ваших хранилищ другими игроками, по умолчанию включено (false)                                                         |
|use-doors          | Защита региона от использования дверей, люков и механизмов их открытия (рычаги, кнопки, плиты и т.п.) от других игроков, по умолчанию включено (false) |

## Установка
1. Скачай последнюю версию мода с GitHub Releases.
2. Убедись, что у тебя установлены Fabric Loader и Fabric API.
3. Помести файл `.jar` в папку `mods` на сервере.
4. После рестарта сервера у вас появится папка `fabricworldguard` в вашей папке `config` сервера. В этой папке будет 3 файла.
5. В файле `config.json` настраиваются лимиты региона и их количество для игрков. Так же доступны кастомные лимты для игрков с префиксами, их необходимо указать в `"groupLimits": {}`. В файле есть подсказка по заполнени.
6. В `flags-config.json` настраиваются динамические флаги. В них можно добавлять или удалять предметы, которые будут разрены для использования дргуими игроками без прав на территории региона. Это временное решени, планируется дальнейшая переработка динамических флагов.
7. В `regions.json` хранятся все созданные регионы на вашем сервере.
